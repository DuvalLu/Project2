# Luis C Velasquez Project 2

## Xena's K9 Paradise - A website for purchasing and obtaining information about different dog breeds. Features a search for keywords that receives and parses the response using JavaScript. Images are displayed on the page from that response. Also, features contact form, css/bootstrap styling, & multiple html pages for breed information. The user will type in specific dog breeds (a list provided for reference to the user) into the search feature and it will return photo examples of that breed. The user can also navigate to the top of the page to view pages specific to breed sizes for additional information and attributes.

## When considering purchasing a dog or looking for general K9 information; visit the website and take some time to click into each respective breed size. Examine their respective attributes. Once you've found the dog you'd like to purchase, fill out a form to get in touch or submit key words to see additional images.
