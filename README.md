# Luis C Velasquez Project 2

## Xena's K9 Paradise - A website for purchasing and obtaining information about different dog breeds. Features a search for keywords that receives and parses the response using javaScript. Images are displayed on the page from that response. Also, features contact form, css/bootstrap styling, & multiple html pages for breed information.

## When considering purchasing a dog or for looking for gerneral K9 information; visit the website and take some time to click into each respective breed size. Examine their respective attributes. Once you've found the dog you'd like to purchase, fill out a form to get in touch or submit key words to see additional images.

## Technologies used:

    1. HTML
    2. CSS
    3. BOOTSTRAP
    4. javaScript
